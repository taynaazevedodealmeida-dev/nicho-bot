# nicho-bot — Documentação de negócio

## O que resolve
PME perde lead no WhatsApp fora do horário. Construtor + simulador de chatbot
pré-treinado por nicho (odonto, estética, advocacia, imobiliária, restaurante,
genérico). Exporta a config em JSON para instalar no cliente.

## Como gera receita
- Setup por nicho (one-off) + mensalidade de manutenção.
- Venda de template pronto.

## Configuração obrigatória
Trocar o WhatsApp `5500000000000` pelo número real no index.html.

## Roadmap
Conectar à WhatsApp Cloud API oficial quando houver receita.
